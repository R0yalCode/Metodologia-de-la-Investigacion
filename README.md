# Analysis of the Use of Autoencoders in Dimensionality Reduction

## Release v1.0 – Initial Literature Review and Extraction Workflow

This repository documents the complete workflow followed during the first phase of the research project focused on Autoencoders and Dimensionality Reduction.

The objective of this release is to provide full methodological traceability and reproducibility of the literature review process conducted using IEEE Xplore and ACM Digital Library.

---

## Research Question

**How do autoencoders contribute to dimensionality reduction?**
